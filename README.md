# 式で練習する暗算トレーナー

ブラウザだけで遊べる日本語の暗算アプリです。「12 + 34」のような式を表示し、指定した秒数がたつと答えを自動で表示します。

## 公開URL

GitHub Pages で公開中です。

<https://ryo-ta8.github.io/-/>

## 機能

- 桁数（1〜4桁）と項の数を調整
- 足し算のみ / 引き算ありを切り替え
- 2秒、3秒、5秒、8秒、10秒から答えまでの時間を選択
- 数秒後に答えを自動表示
- 「できた」「もう一回」で自己採点
- 挑戦回数、連続成功、成功率を表示
- 最近の結果と式を履歴表示

## ローカルで起動

```bash
python3 -m http.server 8000
```

起動後、<http://localhost:8000/> を開いてください。

## デプロイ

このリポジトリは静的サイトとしてそのまま配信できます。GitHub Pages では `.github/workflows/pages.yml` により、`main` ブランチへの push 時にリポジトリルートを公開します。

Netlify では `netlify.toml` により公開ディレクトリが自動設定されます。
