# 給食当番 準備タイマー

小学校の給食当番が時間内に準備できているかを確認するための、ブラウザだけで使えるタイマーアプリです。

## 公開URL

GitHub Pages で公開できます。

- トップページ: <https://ryo-ta8.github.io/->
- わいわい4択クイズ: <https://ryo-ta8.github.io/-/quiz.html>

## 機能

- [4万〜6万円帯 オーブンレンジ比較](microwave.html): 基本性能重視で候補機種を比較し、公式サイト・価格確認リンクへ移動できます

- [自動席替えメーカー](seating.html): 名前を入力して、教室の行・列に合わせたランダムな座席表を作成

- [総当たり対戦順メーカー](match-scheduler.html): 参加チーム名・対戦回数・コート数から、休憩の偏りを抑えた総当たりの試合順を作成
- [学校大画面タイマー](school-timer.html): 電子黒板・プロジェクターで見やすい授業用タイマー。テンプレート、設定保存、大画面表示に対応
- [わいわい4択クイズ](quiz.html): 文章・画像の問題を作成し、電子黒板で4択クイズを出題

- 係ごとの大きなボタンを表示
- 係のボタンを押すと「まだの係」から「準備完了」へ移動
- 押し間違えた場合は「準備完了」のボタンを押すと元に戻せる
- スタート後、設定した準備時間から残り時間を大きくカウントダウン表示
- 1分ごとに約5秒間の急かす音を再生（間隔は設定で変更可能）
- 準備時間を画面上で変更可能
- 係名を読点・カンマ区切りで自由に変更可能

## ローカルで起動

```bash
python3 -m http.server 8000
```

起動後、<http://localhost:8000/> を開いてください。

## デプロイ

このリポジトリは静的サイトとしてそのまま配信できます。GitHub Pages では `.github/workflows/pages.yml` により、`main` ブランチへの push 時にリポジトリルートを公開します。

Netlify では `netlify.toml` により公開ディレクトリが自動設定されます。
