# 給食当番 準備タイマー

小学校の給食当番が時間内に準備できているかを確認するための、ブラウザだけで使えるタイマーアプリです。

## 公開URL

GitHub Pages で公開できます。

<https://ryo-ta8.github.io/-/>

## 機能

- 係ごとの大きなボタンを表示
- 係のボタンを押すと「まだの係」から「準備完了」へ移動
- 押し間違えた場合は「準備完了」のボタンを押すと元に戻せる
- スタート後、経過時間を大きく表示
- 1分ごとに急かす音を再生（間隔は設定で変更可能）
- 係名を読点・カンマ区切りで自由に変更可能

## ローカルで起動

```bash
python3 -m http.server 8000
```

起動後、<http://localhost:8000/> を開いてください。

## デプロイ

このリポジトリは静的サイトとしてそのまま配信できます。GitHub Pages では `.github/workflows/pages.yml` により、`main` ブランチへの push 時にリポジトリルートを公開します。

Netlify では `netlify.toml` により公開ディレクトリが自動設定されます。
